# Controls üïπÔ∏è

- **WASD+mouse** - 3D movement
- **Q/E** - roll the camera
- **Space/Backspace** - move up or down non-relative to the camera
- **Right mouse button** - send distortion to the water
- **R** - show water's vertex grid
- **Escape** - stop registering mouse movement
